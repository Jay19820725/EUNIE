## 2026-03-22 - [Navigation Accessibility]
**Learning:** Adding ARIA labels and title attributes to floating navigation and toggle buttons significantly improves the experience for screen reader users and provides quick visual hints via native tooltips. Consistent focus states using `focus-visible` ensure keyboard navigation is clear and accessible.
**Action:** Always include `aria-label`, `title`, and `focus-visible` rings on icon-only buttons or interactive floating elements.
